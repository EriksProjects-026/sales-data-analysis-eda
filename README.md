# Sales Data Analysis & Price Optimizer

## О проекте

Проект разработан в рамках подготовки к Летней школе Центрального Университета (ЦУ). Идея алгоритма возникла после знакомства с направлениями ЦУ: хотелось решить практическую бизнес-задачу, объединив симуляцию данных, финансовую логику и исследовательский анализ (EDA), а также для реализации нескольких тем из перечня програмы летней школы: Python для анализа и визуализации данных и Поиск информации и работа с гипотезами.

## Цель проекта

Поиск оптимальной стоимости комплектующих (GPU и RAM) для достижения максимальной чистой прибыли. Скрипт не просто моделирует продажи, а рассчитывает точку максимальной маржинальности с учетом факторов реального рынка, выводит интерактивный график прибыли за год и проверяет гипотезы, выводя верные и ложные предположения о продажах.

## Основные возможности

* **Симуляция рыночных условий:** генерация датасета с учетом сезонных пиков (Q4, скидки в январе и августе), агрессивного демпинга конкурентов, нестабильного рынка, инфляции и эластичности спроса.
* **Оптимизация цен:** автоматический поиск ценовой точки, при которой достигается максимальная чистая прибыль по каждой категории товаров.
* **Анализ и визуализация:** построение диаграмм распределения прибыли по месяцам и категориям с помощью Seaborn и Matplotlib.
* **Тестирование гипотез:** автоматизированная проверка 5 бизнес-гипотез с наглядным цветным выводом результатов в консоль.

## Технологический стек

* **Python 3** — базовый язык реализации.
* **random** — симуляция цен конкурентов, скидок и нелинейных зависимостей спроса.
* **pandas** — обработка данных, группировка, фильтрация и расчет показателей чистой выручки, и других различных парраметров, вычисление которых требуется в проверке гипотез.
* **seaborn & matplotlib** — наглядная визуализация результатов и кастомизация графиков.

## Инструкция по запуску

1. Скачать проект или клонировать репозиторий:
   ```bash
   git clone https://github.com/EriksProjects-026/sales-data-analysis-eda.git
   ```

2. Перейти в папку проекта:
   ```bash
   cd sales-data-analysis-eda
   ```

3. Установить зависимости:
   ```bash
   pip install pandas seaborn matplotlib
   ```

4. Запустить скрипт:
   ```bash
   python main.py
   ```
